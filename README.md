# Flight Delay Analysis

## Project

This project analyzes flight delays using machine learning techniques.

## Methods

* PCA (Dimensionality Reduction)
* LDA (Classification Projection)
* DBSCAN & Clustering
* Data Cleaning & Feature Engineering

## Run

Install libraries:

pip install -r requirements.txt

Run notebook:

jupyter notebook notebooks/analysis.ipynb
